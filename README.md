## â“ Troubleshooting

### Can't open the file?
- Make sure you're double-clicking `index.html`
- Try right-click â†’ Open With â†’ Chrome
- Check the file downloaded completely
- Ensure you have the latest version

### Stream won't load?
- Verify the URL is correct
- Check if it's a LIVE stream (not recorded)
- Some platforms block embedding
- Try opening the URL directly first
- Check your internet connection

### Audio not working?
- Click the stream to activate audio
- Only ONE stream plays audio at a time (unless using Audio Mixer)
- Check browser isn't muted
- Some streams don't have audio tracks
- Verify stream is actually live

### Recording issues?

**Recording button disabled:**
- Add at least one stream first
- Check browser permissions for screen recording

**Can't select tab to record:**
- Make sure UniteRev tab is visible
- Try refreshing the page
- Check browser supports Screen Capture API (Chrome/Edge)

**Recording stops unexpectedly:**
- Don't close or minimize the tab during recording
- Ensure enough disk space for download
- Check RAM usage (close other tabs if needed)

**Video file too large:**
- Shorter recordings = smaller files
- Use fewer streams when recording
- WebM compression is automatic

**No audio in recording:**
- Click a stream to activate audio BEFORE starting recording
- Ensure the stream actually has audio
- Check the stream isn't muted

### Scene problems?

**Scenes won't save:**
- Check browser storage isn't full
- Try clearing old scenes you don't need
- Refresh page and try again

**Hotkeys not working:**
- Make sure you're not typing in a text field
- Hotkeys only work when NOT editing text
- Use keys 1-9 (top row, not numpad)

**Scene transitions look wrong:**
- Select different transition type in Scene Manager
- Adjust transition duration slider
- Some transitions may not work in older browsers

### Overlay issues?

**Can't drag overlays:**
- Click and hold to drag
- Make sure overlay panel is open
- Refresh page if stuck

**Overlays don't appear:**
- Check overlay visibility (ğŸ‘ï¸ button)
- Verify overlay has content (text or valid image URL)
- Try adjusting opacity slider

**Image overlay won't load:**
- Verify image URL is correct and accessible
- Image must be publicly accessible (not behind login)
- Try using direct image URLs (ending in .png, .jpg, etc.)
- CORS restrictions may block some images

### Audio Mixer issues?

**No sound from multiple streams:**
- Browser security limits true multi-stream audio
- Use recording feature to capture mixed audio
- Mixer controls preview levels, not actual mixing in browser

**VU meters not moving:**
- Meters are simulated due to browser restrictions
- Volume controls work for recording
- Click streams to test actual audio

### Dashboard not updating?

**Stats stuck or wrong:**
- Refresh the page
- Check browser console for errors
- Clear browser cache

**Layout presets not highlighting:**
- May take a moment to update
- Try clicking a different preset
- Refresh if stuck

### Performance issues?

**Slow or laggy:**
- Reduce number of streams
- Close unnecessary browser tabs
- Use simpler layouts (2Ã—2 instead of 4Ã—4)
- Disable overlays temporarily
- Check RAM usage (need 8GB+ for 9+ streams)

**High memory usage:**
- Close some streams
- Refresh the page periodically
- Use Chrome's Task Manager to identify issues
- Reduce recording quality if available

### Grid looks wrong?
- Click a different layout button
- Refresh the page
- Try clearing browser cache
- Check for conflicting browser extensions

### Lost all settings?

**Streams/scenes disappeared:**
- Check you're using the same browser
- Browser storage may have been cleared
- Try looking in browser's Private/Incognito mode separately
- Settings are per-browser, per-profile
