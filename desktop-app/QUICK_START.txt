üèóÔ∏è  BUILD FOR ALL PLATFORMS
-----------------------------

To build Mac + Windows + Linux all at once:

    cd desktop-app
    npm install
    npm run build:all

Note:
- On Mac, can build Mac + Linux (not Windows)
- On Windows, can only build Windows
- On Linux, can build Linux (not Mac/Windows)

For cross-platform builds, use:
- macOS with Wine (for Windows)
- GitHub Actions (automated builds)


üêõ TROUBLESHOOTING
------------------

"npm install fails":
    npm cache clean --force
    npm install --verbose

"Build fails on Mac":
    xcode-select --install

"Build fails on Windows":
    npm install --global windows-build-tools

"Electron download stuck":
    - Check internet connection
    - Try different network (mobile hotspot)
    - Wait and retry
